Toys API documentation

to get toys

  • to get the first 10 toys:https://omrialter.cyclic.app//toys(GET)
  • to get toys per page(in the example we try page 2): https://omrialter.cyclic.app//toys/?page=2(GET)
  • to sort the results by property(in the example we try to sort by price): https://omrialter.cyclic.app//toys/?sort=price(GET)
  • to reverse the order of the results (in the example we reverse the results of the price sort): https://omrialter.cyclic.app//toys/?sort=price&&reverse=yes(GET)
  • to search for results by name of toy or by the information of the toy(in the example we try to search a toy with the word pazel in the name or info ): https://omrialter.cyclic.app//toys/search/?s=pazel(GET)
  • to get a list of toys from a specific category(lets say sport category): https://omrialter.cyclic.app//toys/category/?s=sport(GET)
  • to get a single toy by his id (in this example you need to change the "toy_id" with the actual id of the toy): https://omrialter.cyclic.app//toys/single/toy_id(GET)

to post a new toy

in order to post a toy you must log in with a user and you must send a valid token

  • to create a new toy: https://omrialter.cyclic.app//toys(POST)

the schema to create a new toy:
name: String,
info: String,
category: String,
img_url: String,
price: Number

to edit a toy

in order to edit a toy property you need to send a valid token

  • to edit a toy(in this example you need to change the "toy_id" with the actual id of the toy): https://omrialter.cyclic.app//toys/toy_id(PUT)

to delete a toy

in order to delete a toy you must send a valid token

  • to delete a toy(in this example you need to change the "toy_id" with the actual id of the toy): https://omrialter.cyclic.app//toys/toy_id(DELETE)

notice that you send an empty json in the body request: "{}"

Sign up

  • to sign up: https://omrialter.cyclic.app//users(POST)

the schema to create a new user:
name: String,
email: String,
password: String

Log in:

  • to log in: https://omrialter.cyclic.app//users/login(POST)

the schema to log in:
email: String,
password: String
